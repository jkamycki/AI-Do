# Mobile App Assets

Shared native mobile assets for the A.I Do WebView app.

- `aido-logo.png` - Native top bar and splash logo source.
- `splash-logo.png` - Splash screen logo source.
- `app-icon-source.png` - Source image for final app icon exports.

The Android and iOS projects also include platform-local copies under `mobile/android/app/src/main/res` and `mobile/ios/AIDoWedding/Assets.xcassets`.
